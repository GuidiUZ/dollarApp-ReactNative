# Dollar Exchange Rate App - React Native

## Overview

This React Native application is designed to provide users with up-to-date information about the exchange rates of the US Dollar against various currencies. Users can view the current exchange rates, select different currencies, and calculate conversions based on the latest data.

## Features

- **Live Exchange Rates**: Fetches and displays real-time exchange rates for the US Dollar against multiple currencies.
- **Currency Selection**: Allows users to choose different currencies to view their exchange rates.
- **Conversion Calculator**: Provides a currency conversion calculator for users to calculate the equivalent amount in different currencies.
- **Responsive Design**: Utilizes React Native for cross-platform development with a responsive user interface.

## Folder Structure

The project's source code is organized into several key folders:

- `components`: Contains reusable React components used throughout the application.
- `hooks`: Includes custom React hooks for managing state and logic.
- `navigation`: Manages the navigation structure of the app using React Navigation.
- `screens`: Defines the main screens of the application, including the home screen and the screen for currency conversion.

## Usage

To use this application, follow these steps:

1. Clone the repository:

```bash
git clone https://github.com/GuidiUZ/dollarApp-ReactNative.git
```

2. Navigate to the `src` folder:

```bash
cd dollarApp-ReactNative/src
```

3. Install dependencies:

```bash
npm install
```

4. Run the application:

```bash
npm start
```

This will start the development server, and you can use an emulator or a physical device to view the application.

## Dependencies

- React
- @react-navigation/native
- react-native
- @rneui/themed (Assuming this is a component library for React Native)

Note: Make sure to install these dependencies using a package manager like npm or yarn.

# Aplicación de Tasa de Cambio del Dólar - React Native

## Descripción general

Esta aplicación React Native está diseñada para proporcionar a los usuarios información actualizada sobre las tasas de cambio del dólar estadounidense frente a varias monedas. Los usuarios pueden ver las tasas de cambio actuales, seleccionar diferentes monedas y realizar conversiones basadas en los datos más recientes.

## Características

- **Tasas de Cambio en Tiempo Real**: Obtiene y muestra tasas de cambio en tiempo real para el dólar estadounidense frente a varias monedas.
- **Selección de Monedas**: Permite a los usuarios elegir diferentes monedas para ver sus tasas de cambio.
- **Calculadora de Conversión**: Proporciona una calculadora de conversión de moneda para que los usuarios calculen el monto equivalente en diferentes monedas.
- **Diseño Responsivo**: Utiliza React Native para el desarrollo multiplataforma con una interfaz de usuario receptiva.

## Estructura de Carpetas

El código fuente del proyecto está organizado en varias carpetas clave:

- `components`: Contiene componentes React reutilizables utilizados en toda la aplicación.
- `hooks`: Incluye hooks personalizados de React para gestionar el estado y la lógica.
- `navigation`: Gestiona la estructura de navegación de la aplicación utilizando React Navigation.
- `screens`: Define las pantallas principales de la aplicación, incluida la pantalla de inicio y la pantalla para la conversión de moneda.

## Uso

Para utilizar esta aplicación, sigue estos pasos:

1. Clona el repositorio:

```bash
git clone https://github.com/GuidiUZ/dollarApp-ReactNative.git
```

2. Navega a la carpeta `src`:

```bash
cd dollarApp-ReactNative/src
```

3. Instala las dependencias:

```bash
npm install
```

4. Ejecuta la aplicación:

```bash
npm start
```

Esto iniciará el servidor de desarrollo, y puedes usar un emulador o un dispositivo físico para ver la aplicación.

## Dependencias

- React
- @react-navigation/native
- react-native
- @rneui/themed (asumiendo que es una biblioteca de componentes para React Native)

Nota: Asegúrate de instalar estas dependencias utilizando un gestor de paquetes como npm o yarn.
